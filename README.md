# School Portal

A web application for managing school operations across Primary and Secondary levels.  
This portal streamlines academic and administrative processes, providing a centralized platform for students, teachers, and administrators.

---

## âœ¨ Features
- **Role-Based Authentication**: Secure login for Admin, Teacher, and Student.
- **Student Profiles**: Manage student information (name, age, class, profile picture).
- **Results Management**: Upload and view scores (Mid-term + Exam + Total).
- **Fees Tracking**: Monitor total, paid, and outstanding balances.
- **Notes Upload/Download**: Teachers upload study materials, students download.
- **Report Cards**: Generate downloadable PDF report cards with remarks from Class Teacher and Head Teacher.
